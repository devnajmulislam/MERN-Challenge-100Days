### 1. Mathematical Operations & Math Object
- Numbers niye kaj kora: Positive, Negative, Float, Odd/Even, Max, Min numbers.
- Basic Arithmetic: `+`, `-`, `*`, `/`, `%` (Remainder), Average nirnoy.
- **Math Object:** `Math.max()`, `Math.min()`, `Math.PI`, `Math.abs()`, `Math.round()`, `Math.floor()`, `Math.ceil()`, and `Math.random()`.

### 2. Variable & Data Types
- `let`, `const`, and `var`-er moddhe parthokko bujhte para.
- **Primitive Types:** Number, String, Boolean, Undefined, Null.
- **Non-Primitive Types:** Array, Function, Object.

### 3. Array Manipulation & Methods
- **Basic:** `indexNo`, `length`, `push`, `pop`, `shift`, `unshift`.
- **Advanced:** `includes`, `indexOf`, `Array.isArray()`, `join`, `concat`, `slice`, `splice`.
- **Sorting:** `sort`, `[...Array].sort` (original array thik rekhe sort kora), ebong `reverse`.

### 4. Control Flow & Loops
- **Conditionals:** `if...else`, `if...else if()...else, logical operators (`>`, `<`, `>=`, `<=`,` === `,` !==`,`&&`, `||`).
- **Loops:** `for of`, `for `, `while`, `for`, `do while`.
- **Loop Control:** `break` and `continue`.
- **Object Iteration:** `for in` loop bebohar kore object scan kora.

### 5. String, Object & Functions
- **String:** `typeof`, `lowerCase`, `upperCase`, `trim`, `slice`, `split`, `join`, `concat`.
- **Object:** Dot notation vs Bracket notation, `Object.keys`, `Object.values`, `delete`.
- **Function:** Parameter passing, arguments handle kora, and function theke `return` neya.
- **Extra:** `new Date()` methods and Array destructuring `[x, y] = [4, 5]`.
